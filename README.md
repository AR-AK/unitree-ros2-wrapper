# ROS2 Wrapper of Unitree SDK for Go1 Robot

# About
This is a ROS2 Wraper package to replace the developer api for the no-longer supported Unitree_Legged_SDK. Streamlined to use modern CPP instead of Boost, and focuses only on motion.

# Dependencies
- ROS2 (desktop full)

- udev
```
# In terminal:
sudo apt update
sudo apt install libudev-dev
```
